# DHCP

## DHCP原理

1. 客户端发送广播包
2. 服务器提供响应，在自己的数据库中查找可用地址交付（同样是广播包）
3. 客户端确认IP，服务器确认地址已分配
4. 服务器确认租约
5. 当客户机重启和租约到达一半时，进行续约；租期间无法联系，租约到达后直接重新发送广播包
